# MyAiProj - 个人工具网站项目总览

## 1. 项目简介

一个简洁优雅的**个人博客主页**，包含实用的在线工具集。项目采用纯前端静态页面架构，无需后端服务器，支持快速部署和访问。

**核心价值**：打造个人品牌形象，展示技术能力，提供实用工具入口。

## 2. 项目组成

### 2.1 主页 (index.html)
- 个人博客主页入口
- 展示所有工具的导航页
- 个人介绍和联系方式

### 2.2 工具集

| 工具名称 | 文件路径 | 功能描述 |
|----------|----------|----------|
| 安全密码生成器 | tool/HTML_密码生成器/HTML_密码生成器.html | 生成高强度密码，支持自定义长度、字符类型、密码强度检测、二维码生成 |
| 工作日期计算器 | tool/HTML_工作日计算器/HTML_工作日计算器.html | 精确计算项目交付日期，支持多种排班模式和法定假期配置 |

## 3. 技术架构

### 3.1 技术栈
- **前端框架**：原生 HTML5 + CSS3 + JavaScript
- **样式方案**：Tailwind CSS 3 (CDN)
- **字体**：Noto Sans SC (Google Fonts)
- **图标**：Emoji + Inline SVG
- **构建工具**：无需构建，静态部署

### 3.2 目录结构
```

├── index.html                          # 个人博客主页
```

## 4. 设计规范

### 4.1 色彩方案
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色调 | #667eea | 按钮、链接、强调元素 |
| 辅助色 | #764ba2 | 渐变结束色 |
| 背景色 | #F8FAFC | 页面背景 |
| 卡片背景 | #FFFFFF | 组件卡片 |
| 文字主色 | #1F2937 | 主要文字 |
| 文字辅助 | #6B7280 | 次要文字 |

### 4.2 字体
- **标题**：Noto Sans SC Bold
- **正文**：Noto Sans SC Regular
- **等宽**：系统默认等宽字体

### 4.3 圆角与阴影
- 卡片圆角：12px - 16px
- 按钮圆角：8px - 12px
- 卡片阴影：`0 4px 20px rgba(0, 0, 0, 0.06)`
- 悬停阴影：`0 10px 30px rgba(0, 0, 0, 0.08)`

### 4.4 动画效果
- 过渡时长：0.2s - 0.3s
- 悬停上移：translateY(-2px)
- 卡片悬停：阴影加深 + 上移效果

## 5. 响应式设计

| 设备类型 | 断点 | 布局 |
|----------|------|------|
| 桌面端 | ≥ 1024px | 最大宽度1000px，居中显示 |
| 平板端 | 768px - 1023px | 自适应布局 |
| 移动端 | < 768px | 全宽显示，垂直堆叠 |

## 6. 部署说明

### 6.1 部署方式
- **静态文件部署**：无需服务器端渲染
- **服务器要求**：支持静态HTML文件的Web服务器
- **推荐方案**：Nginx、Apache、Vercel、Netlify、GitHub Pages

### 6.2 注意事项
1. 工具页面使用相对路径，确保目录结构不变
2. 依赖CDN资源，需要网络连接
3. 跨域问题：工作日期计算器的假期API需要CORS支持
